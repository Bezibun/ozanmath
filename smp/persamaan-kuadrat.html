Mengenal Dasar

Apa itu Bentuk Umum?

Persamaan kuadrat adalah persamaan matematika dengan pangkat tertinggi variabelnya adalah dua. Untuk menyelesaikannya, kita harus mengubahnya ke Bentuk Baku:

ax² + bx + c = 0

a

Koefisien x²: Angka di depan x². Syarat utama: a ≠ 0.

b

Koefisien x: Angka di depan variabel x.

c

Konstanta: Angka yang berdiri sendiri tanpa variabel.

Contoh Identifikasi:

3x² + 2x - 5 = 0

a=3, b=2, c=-5

x² - 9 = 0

a=1, b=0, c=-9

Eksperimen Persamaan Kamu:

1x² + 5x + 6 = 0

3 Strategi Utama

Pilih metode yang paling sesuai dengan tingkat kesulitan soalmu.

01

Faktorisasi

Cara tercepat jika nilai a=1 dan angka-angkanya bulat. Prinsipnya adalah mencari dua angka yang pas.

Logika Angka

Cari p dan q dimana:
p × q = c
p + q = b

Langkah 1

x² + 5x + 6 = 0

Cari angka yang dikali=6, dijumlah=5. Jawab: 2 dan 3.

Langkah 2

(x + 2)(x + 3) = 0

Masukkan angka tadi ke dalam kurung faktor.

Hasil

x = -2 atau x = -3

02

Melengkapkan Kuadrat

Digunakan saat soal sulit difaktorkan tapi kamu ingin menghindari rumus yang panjang.

Aturan Utama

Tambahkan (b/2)² ke kedua ruas persamaan agar sisi kiri bisa menjadi kuadrat sempurna.

Langkah 1

x² + 6x = -5

Pindahkan konstanta ke kanan.

Langkah 2

x² + 6x + (3)² = -5 + 9

Tambah (6/2)² = 9 di kedua ruas.

Langkah 3

(x + 3)² = 4 → x + 3 = ±2

Akarkan kedua ruas.

03

Rumus ABC

Senjata pamungkas. Berhasil untuk semua jenis persamaan kuadrat, termasuk yang hasilnya akar atau desimal.

Rumus Kuadratik

x = -b ± √(b² - 4ac)
2a

Ingat: b² - 4ac disebut Diskriminan. Jika hasilnya negatif, maka tidak ada akar real.

Pembahasan Soal

Soal: 2x² - 5x - 3 = 0

1. Identifikasi: a=2, b=-5, c=-3

2. Diskriminan:
D = (-5)² - 4(2)(-3) = 25 + 24 = 49

3. Hitung x:
x = [5 ± √49] / 2(2)
x = [5 ± 7] / 4

x₁ = (5+7)/4 = 3

x₂ = (5-7)/4 = -0.5

Tes Pemahaman

SOAL 1 / 5
