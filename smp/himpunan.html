Teori Himpunan

Panduan lengkap memahami objek, simbol, dan operasi matematika himpunan dengan cara yang mudah dipahami.

1. Pengertian & Contoh

Himpunan adalah kumpulan objek atau benda yang memiliki keterangan/definisi yang jelas (well-defined).

Contoh Himpunan:

  • Himpunan warna pelangi
  • Himpunan bilangan asli kurang dari 10
  • Himpunan nama hari yang berawalan 'S'

Bukan Himpunan:

  • Kumpulan orang tinggi (Tinggi itu relatif)
  • Kumpulan lukisan bagus (Bagus itu subjektif)
  • Kumpulan siswa pandai

2. Cara Menyatakan Himpunan

Ada tiga cara utama untuk menyatakan suatu himpunan. Mari kita ambil contoh: Himpunan bilangan ganjil antara 1 sampai 10.

A

Deskripsi (Kata-kata)

Menuliskan syarat keanggotaan dengan kalimat.

A = { Bilangan ganjil antara 1 dan 10 }

B

Mendaftar Anggota (Enumerasi)

Menyebutkan semua anggotanya satu per satu di dalam kurung kurawal.

A = { 3, 5, 7, 9 }

C

Notasi Pembentuk Himpunan

Menggunakan variabel (biasanya x) diikuti syarat batasnya.

A = { x | 1 < x < 10, x ∈ Bilangan Ganjil }

(Dibaca: x sedemikian hingga x lebih besar dari 1 dan lebih kecil dari 10, di mana x adalah anggota bilangan ganjil)

3. Simbol & Istilah Penting

Simbol Arti Contoh Penggunaan
∈ Anggota Himpunan (Elemen) 2 ∈ {Bilangan Genap}
∉ Bukan Anggota Himpunan 3 ∉ {Bilangan Genap}
n(A) Kardinalitas (Jumlah Anggota) A={1,2}, maka n(A) = 2
∅ atau { } Himpunan Kosong {Nama hari berawalan 'Z'}
S Himpunan Semesta Seluruh objek yang dibicarakan

4. Operasi Himpunan

Misalkan: A = { 1, 2, 3, 4 } dan B = { 3, 4, 5, 6 }

Irisan (Intersection - ∩)

Anggota yang ada di A DAN ada di B.

A ∩ B = { 3, 4 }

Gabungan (Union - ∪)

Semua anggota A ATAU anggota B digabung.

A ∪ B = { 1, 2, 3, 4, 5, 6 }

Selisih (Difference -)

Anggota A yang BUKAN anggota B.

A - B = { 1, 2 }

Komplemen (Ac)

Anggota S yang BUKAN anggota A (Misal S={1..10}).

Ac = { 5, 6, 7, 8, 9, 10 }

5. Himpunan Bagian (Subset)

Himpunan A dikatakan himpunan bagian dari B (ditulis A ⊂ B) jika setiap anggota A juga merupakan anggota B.

Contoh:

A = { 1, 2 }

B = { 1, 2, 3, 4, 5 }

Maka: A ⊂ B (A adalah bagian dari B)

Note: Jumlah seluruh himpunan bagian dari sebuah himpunan dengan n anggota adalah 2^n.

Coba Nyatakan ke Enumerasi!

Ubah notasi { x | x < 10, x ∈ Bilangan Prima } menjadi daftar anggota.
