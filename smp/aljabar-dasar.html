Mulai Belajar Uji Kemampuan

1. Mengenal Unsur Aljabar

"Anatomi dasar yang harus kamu pahami."

Contoh Bentuk Aljabar

2x + 5

Variabel

Simbol huruf mewakili angka. Contoh: x.

Koefisien

Angka pengali variabel. Contoh: 2.

Konstanta

Angka tetap tanpa variabel. Contoh: 5.

Suku

Dipisahkan + atau -. Contoh: 2x dan 5.

2. Penjumlahan & Pengurangan

Aturan Emas: Gunakan Suku Sejenis.

Penjumlahan

3x + 2x = 5x

3 apel + 2 apel = 5 apel.

Pengurangan

10y - 4y = 6y

Hanya kurangi koefisien variabel yang sama.

3. Sifat-Sifat Aljabar

Memahami "cara main" dalam menyederhanakan bentuk matematika.

Komutatif

"Pertukaran tempat tidak mengubah hasil akhir."

Rumus Umum

a + b = b + a

Contoh Bertahap

  • 1. Misal ada 3x + 5y
  • 2. Tukar posisi sukunya.
  • 3. Menjadi 5y + 3x
  • Keduanya bernilai sama!

Asosiatif

"Pengelompokan suku tidak mengubah hasil akhir."

Rumus Umum

(a + b) + c = a + (b + c)

Contoh Bertahap

  • 1. Misal: (2x + 3x) + 4x
  • 2. Kerjakan kurung: 5x + 4x = 9x
  • 3. Coba ubah kurung: 2x + (3x + 4x)
  • 4. Hasil tetap: 2x + 7x = 9x

Distributif

"Penyebaran perkalian ke dalam penjumlahan."

Rumus Umum

a(b + c) = ab + ac

Contoh Bertahap

  • 1. Soal: 3(x + 4)
  • 2. Kalikan 3 dengan x: 3 * x = 3x
  • 3. Kalikan 3 dengan 4: 3 * 4 = 12
  • 4. Gabungkan: 3x + 12
